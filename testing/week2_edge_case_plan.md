# Week 2 Edge Case Plan

## Authentication Edge Cases
- Invalid phone number
- Invalid DOB
- Wrong verification answers
- Authentication failure after 3 attempts

## Policy Edge Cases
- No active policy found
- Expired policy
- Invalid selected policy

## Claim Edge Cases
- No claims under selected policy
- Invalid claim ID
- Claim already closed

## API Edge Cases
- API timeout
- API error
- Empty response
- Invalid response format

## Conversation Edge Cases
- User gives unrelated input
- User remains silent
- User asks for human support repeatedly
- User changes intent mid-conversation
