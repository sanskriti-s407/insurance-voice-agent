## Agent Name

Insurance Voice Agent

## Purpose

The Main Agent acts as the central orchestration layer of the solution.

It is responsible for:

* Customer authentication
* Intent detection
* Context management
* Session management
* Multi-policy handling
* Customer onboarding
* Subagent invocation
* Customer update request management
* Escalation handling

## Responsibilities

1. Authenticate customer.
2. Maintain session context.
3. Route customer requests to appropriate subagents.
4. Support multiple policies for a single customer
5. Handle fallback scenarios.
6. Manage escalation requests.
7. Transfer conversation context during agent handoff.

## Supported Customer Requests

* New Policy Purchase
* Product Coverage Information
* Multi-Policy Inquiry
* Policy Details
* Benefits Information
* Claim Status
* New Claim Initiation
* Policy Renewal
* Customer Update Request
* Agent Escalation
