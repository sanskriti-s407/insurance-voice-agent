## Agent Name

Insurance Voice Agent

## Purpose

The Main Agent acts as the orchestration layer of the solution.

It is responsible for:

* Customer authentication
* Intent detection
* Context management
* Session management
* Subagent invocation
* Escalation handling

## Responsibilities

1. Authenticate customer.
2. Maintain session context.
3. Route customer requests to appropriate subagents.
4. Handle fallback scenarios.
5. Manage escalation requests.
6. Transfer conversation context during agent handoff.

## Supported Customer Requests

* New Policy Purchase
* Policy Inquiry
* Benefits Information
* Claim Status
* New Claim Initiation
* Policy Renewal
* Agent Escalation
