# Session Parameters

| Parameter | Purpose |
|---|---|
| phone_number | Stores registered mobile number |
| dob | Stores date of birth |
| verification_answer_1 | Stores first verification answer |
| verification_answer_2 | Stores second verification answer |
| auth_status | Stores authentication status |
| retry_count | Stores authentication retry count |
| customer_id | Stores authenticated customer ID |
| customer_name | Stores customer name |
| policy_list | Stores policies linked to customer |
| selected_policy_id | Stores policy selected by customer |
| claim_list | Stores claims linked to selected policy |
| selected_claim_id | Stores claim selected by customer |
| product_type | Stores selected insurance product type |
| otp | Stores OTP entered by new user |
| vehicle_number | Stores vehicle registration number |
| chassis_number | Stores chassis number |
| escalation_reason | Stores reason for escalation |
| ticket_id | Stores generated ticket ID |
| conversation_summary | Stores summary for live agent handoff |
